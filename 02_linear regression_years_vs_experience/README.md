# 📈 Linear Regression: Salary vs Years of Experience

This project demonstrates how **Simple Linear Regression** can be used to model the relationship between **Years of Experience** and **Salary**. The goal is to understand how salary grows with experience and evaluate how well a linear model fits the data.

---

## 🎯 Project Objective

To build a regression model that predicts an employee’s **salary** based on their **years of professional experience** and evaluate model performance using residual analysis.

---

## 📊 Dataset Description

The dataset contains two variables:

| Feature | Description |
|---------|-------------|
| YearsExperience | Number of years of professional experience |
| Salary | Annual salary corresponding to the experience |

This is a classic regression problem with a **strong positive linear relationship**.

---

## 🧪 Steps Performed

1. Imported and explored the dataset  
2. Visualized the relationship between experience and salary  
3. Trained a **Simple Linear Regression** model  
4. Predicted salaries using the trained model  
5. Evaluated model performance using residual analysis  
6. Visualized salary distribution
